# 🎮 兒童學習樂園 — 遊戲說明

## 簡介
「兒童學習樂園」是一個專為學齡前與低年級兒童設計的互動學習網頁，採用繁體中文介面，透過可愛的 Emoji 與活潑的視覺風格，讓孩子在遊戲中快樂學習。

## 遊戲模式

### 1. 🔢 數數字
- 畫面會顯示一排 Emoji 圖案（例如 🍎🍎🍎）
- 請數一數有幾個，然後選出正確的數字
- 適合訓練觀察力與基礎數數能力

### 2. ➕ 加法
- 畫面會顯示加法算式與對應的圖案數量
- 計算出答案後選擇正確數字
- 適合練習 20 以內的簡單加法

### 3. 🐶 認動物
- 畫面會顯示一個動物 Emoji
- 請從四個選項中選出正確的動物名稱
- 共有 10 種可愛動物，適合認識動物與國字

## 遊玩方式

1. 上方點選遊戲模式切換（數數字 / 加法 / 認動物）
2. 每回合有 10 道題目
3. 點擊選項作答，正確得 10 分
4. 點選「下一題」繼續挑戰
5. 完成 10 題後顯示成績與鼓勵訊息
6. 點「再玩一次」重新挑戰

## 計分與回饋

- 每題答對得 10 分，答錯不扣分
- 答對時按鈕變綠色，顯示 ✅ 鼓勵提示
- 答錯時按鈕變紅色，並顯示正確答案
- 全部答對會觸發彩色紙花動畫 🎉

## 技術說明

- 單一 HTML 檔案，無需安裝或網路連線
- 純前端 JavaScript，無後端依賴
- 支援手機與電腦瀏覽器
- 使用 CSS 動畫與 RWD 響應式設計
